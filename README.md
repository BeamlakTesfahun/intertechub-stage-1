## Steps to run locally

1. Clone the repository:

```bash
   git clone https://github.com/BeamlakTesfahun/intertechub-stage-1.git
   cd intertechub-stage-1
```

2. Install dependencies:

```bash
   npm install
```

3. Create a .env file and specify the port

```env
   PORT=3000
```

4. Start the server:

```bash
   npm run dev
```
